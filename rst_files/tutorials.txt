=============================
Phenix Examples and Tutorials
=============================


`Video tutorials <https://www.youtube.com/c/phenixtutorials>`__ with data
=========================================================================

The data for each of these tutorials are available in the GUI by selecting 'New Project'
and then 'Set up Tutorial Data' and selecting a tutorial.  Each video tutorial provides some background and then carries out a demonstration using the tutorial data.
