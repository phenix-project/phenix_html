CryoFit2: Fitting to a Cryo-EM Map using phenix.dynamics
==================================================================

.. contents::

Overview
----------
Unlike CryoFit1 that uses gromacs, CryoFit2 runs with phenix files
only. Therefore it doesn't require gromacs installation and is faster to execute.
