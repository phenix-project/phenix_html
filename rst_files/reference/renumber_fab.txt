phenix.renumber_fab
===================

Description
-----------

This tool is for renumbering residues in FAB chaings according to appropriate
scheme. Supported schemes are Kabat, Chothia and Martin.

Requirements: model and sequence.

Sequence should contain the full sequence of the FAB and it  should at
least cover whole chains in the model.

!!! WARNING !!! This program sends supplied sequence to AbNum server at
http://www.bioinf.org.uk/abs/abnum/
Please acknowledge the server authors:
Abhinandan KR, Martin AC. Mol Immunol. 2008 Aug; 45(14):3832-9.
https://doi.org/10.1016/j.molimm.2008.05.022


Command line usage examples
---------------------------

  % phenix.renumber_fab model.pdb sequence.fa

  % phenix.renumber_fab 2jjj.cif sequence.fa scheme=chothia


GUI
---

This tool is available in the Phenix GUI in the "PDB deposition" section.
